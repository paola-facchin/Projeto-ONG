# Descrição do Projeto

O site ONG Caminhar Juntos(Ficticia) é uma plataforma digital voltada para divulgar as ações da ONG, apresentar projetos, facilitar o cadastro de voluntários, informar sobre doações e transparência, além de oferecer um canal de comunicação direto com o público. O objetivo principal é conectar pessoas interessadas em ajudar, fortalecendo a comunidade por meio do voluntariado e do apoio social.

# Funcionalidades Principais

- Navegação intuitiva com menu principal para acesso rápido às seções.
- Formulários completos e validados.
- Apresentação de projetos, notícias e eventos da ONG.
- Seção dedicada para informações sobre doações e transparência financeira.
- Canal de contato para dúvidas e suporte.
- Responsividade para dispositivos móveis e desktops.
- Uso de vídeo e galerias para maior engajamento visual.

## 👥 Personas Atendidas

- Administradores da ONG
- Voluntários
- Doadores
- Visitantes

## 📚 Tecnologias Utilizadas

- HTML5 (semântica, formulários, multimídia, gráficos)
- CSS3 (responsivo, design adaptável)
- JavaScript (gráficos interativos, máscaras de input)
- Google Fonts (Inter) para tipografia moderna
- Ferramenta: W3C Validator
