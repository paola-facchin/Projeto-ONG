# 📝 Especificação de Formulários

## 1. Cadastro de Voluntário (voluntariado.html)
- Nome, Email, CPF, Telefone, CEP (com máscaras)
- Habilidades (select), Dias disponíveis (checkboxes)
- Agrupado com `<fieldset>`
- Validações nativas HTML5

## 2. Inscrição em Projeto (projetos.html)
- Nome, Email, Experiência (select), Motivação (textarea)
- Campos obrigatórios
- Formulário simples, limpo e acessível

## 3. Contato Institucional (contato.html)
- Assunto, Mensagem, Anexo (accept: pdf, doc, img)
- Textarea com limite de linhas
- Atributo `required` nos campos principais

## 4. Newsletter (blog.html / rodapé)
- Email e seleção múltipla de preferências
- Combinação de input `email` e `select multiple`
